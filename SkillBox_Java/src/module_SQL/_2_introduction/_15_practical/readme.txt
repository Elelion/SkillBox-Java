Язык запросов SQL
Вы увидели, что из себя представляет язык запросов SQL и как эти запросы
выполняются. Предлагаем выполнить несколько SQL-запросов к базе данных
MySQL в инструменте, которым вы решили пользоваться (PHPMyAdmin или Workbench).

Выполните следующие запросы и убедитесь, что результат их выполнения
соответствует приведённым ниже скриншотам.

Запрос 1. Получение товаров дороже 800 рублей

SELECT *
FROM `good`
WHERE `price` > 800;

Запрос 2. Получение списка закончившихся товаров

SELECT *
FROM `good`
WHERE `count` = 0;

Или так

SELECT * FROM SkillBox_Shop.good WHERE price > 100 AND count = 0;

Запрос 3. Получение общего количества оплаченных заказов

Примечание: оплаченные заказы — это заказы, у которых status_id равен 7.

SELECT COUNT(*) FROM `order` WHERE `status_id` = 7;
