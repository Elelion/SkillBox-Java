В предыдущих видео вы познакомились с языком запросов SQL и увидели, как эти
запросы выполняются.

данных shop.sql:

Получите список товаров дороже 800 рублей.
Получите список закончившихся товаров.
Получите общее количество оплаченных заказов (оплаченные заказы — это заказы,
